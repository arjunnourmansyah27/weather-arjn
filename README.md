# 🌤 Arjn Weather App

Arjn Weather App adalah aplikasi web sederhana yang memungkinkan pengguna mencari informasi cuaca suatu kota secara real-time. Aplikasi ini menggunakan **OpenWeather API** untuk mendapatkan data cuaca terkini.

## 🚀 Fitur
- 🔍 Pencarian cuaca berdasarkan nama kota.
- 🌡️ Menampilkan suhu dalam satuan Celcius.
- 🌦️ Menampilkan deskripsi cuaca dan ikon kondisi saat ini.
- 🌬️ Menampilkan kecepatan angin.
<!-- - 🗓️ Menampilkan tanggal dan hari saat ini. -->
- ⏳ Indikator loading saat data sedang diambil.
- ⚠️ Notifikasi jika kota tidak ditemukan.

## 🛠️ Teknologi yang Digunakan
- **React.js** - Framework untuk membangun antarmuka pengguna.
- **Axios** - Untuk mengambil data dari OpenWeather API.
- **React Loader Spinner** - Untuk menampilkan animasi loading.
- **FontAwesome** - Untuk ikon notifikasi error.

## 🌎 API yang Digunakan
Aplikasi ini memanfaatkan **OpenWeather API**, yang dapat diakses di:
🔗 [OpenWeather API](https://openweathermap.org/api)
